NF-e automatica pela chave

Ao bipar a chave, o sistema pode abrir a consulta publica ja com a chave para voce so resolver o reCAPTCHA
Fluxo principal: ler a chave e buscar os dados oficiais na Receita/DF-e ou importar um unico arquivo XML/PDF/HTML para revisar os itens antes do lancamento.
Alternativa manual: fotografe a grade dos itens e use a imagem como referencia visual para digitar codigo, descricao, quantidade e valor com mais rapidez.
Se voce enviar um PDF ou HTML da consulta publica e ele tiver a chave, o sistema tenta aproveitar os dados de contingencia. So cai na leitura aproximada quando o DF-e nao estiver disponivel.

        FreePBX

        Use este modo para registrar os usuários no seu FreePBX.
        Carregando configuração SIP... Use os placeholders {nome}, {login} e {ramal}. Exemplo: {nome} RioBranco Para funcionar no navegador, o PABX/SBC precisa expor SIP sobre WebSocket seguro (WSS) com suporte a WebRTC. Padrao recomendado: importar apenas a CA interna do RioBranco. Windows e Linux: use o script. Firefox ESR/TinyCore: use o script Linux com o navegador fechado. iPhone/iPad: use o perfil .mobileconfig. Android: use "Baixar CA Android" (.crt) e importe como Certificado CA. O P12/PFX pode ser recusado porque nao contem chave privada. Bundles PKCS#12/PFX: senha padrao riob1951. Quando a CA estiver configurada, o bundle contem a CA; sem CA, cai em fallback para os certificados publicos do servidor.

        NF-e / Receita

        Carregando configuração NF-e... Diagnostico da integracao NF-e indisponivel. O fluxo oficial DF-e usa certificado digital do destinatario para baixar XML pela chave e por NSU. Azure Document Intelligence: configure endpoint e chave para habilitar a leitura dos itens por foto. Modo portal assistido: ao bipar a chave, o sistema abre a consulta oficial para voce resolver o reCAPTCHA. Depois, use o XML oficial para lancar automaticamente. Modo certificado digital: usa o fluxo oficial NFeDistribuicaoDFe para buscar XML pela chave impressa no DANFE e tambem permite sincronizar novas notas por NSU.
